Derrubando o Primeiro Colosso: O Ambiente de Desenvolvimento

Neste post conto como enfrentei o desafio de instalar o VS Code e configurar meus primeiros arquivos HTML para o portfólio.

Ler Artigo
O Desafio Inicial Assim como Wander chega às Terras Proibidas sem saber exatamente o que o espera, iniciei meu projeto de blog com um objetivo claro, mas muitos desafios técnicos pela frente. O primeiro passo não foi uma batalha de espada, mas sim a configuração da minha "arma": o VS Code. A Estratégia Instalar o editor e entender como estruturar os primeiros arquivos HTML foi como aprender a escalar o primeiro colosso, Valus. Era preciso paciência e atenção aos detalhes para que a estrutura básica ficasse firme e funcional. A Preparação: Baixei o VS Code e criei o meu repositório no GitHub para garantir que cada progresso fosse registrado. O Primeiro "Golpe": Escrever as primeiras linhas de código no arquivo ⁠index.html⁠ e vê-lo renderizar no navegador foi a minha primeira vitória. O Caminho à Frente Este é apenas o começo. Com o ambiente de desenvolvimento pronto, agora sinto que tenho as ferramentas necessárias para dar vida ao design épico de Shadow of the Colossus. O site já está conectado ao Vercel e o "campo de batalha" está preparado para as próximas etapas de estilização. Dica do Manu: Como você é uma pessoa criativa e gosta de desenhar, você pode adicionar uma imagem de um rascunho seu do primeiro colosso ou um print do seu código no VS Code bem organizado no meio desse texto. Isso deixa o post com a sua cara!

A Luz da Espada: Aprendendo CSS

Assim como Wander usa a espada para achar o caminho, usei o CSS para iluminar e dar cor ao meu site escuro.

Ler Artigo
O Poder da Iluminação Nas Terras Proibidas, a espada de Wander não serve apenas para o combate; ela é a guia, a fonte de luz que revela o caminho através das sombras. No desenvolvimento do meu blog, o CSS desempenhou exatamente esse papel. Minha Estratégia Depois de estruturar a base com o HTML, o site estava apenas um esqueleto bruto. O CSS foi a ferramenta que usei para "iluminar" o projeto: Definindo a Atmosfera: Escolhi paletas de cores que remetem à melancolia e grandiosidade do jogo, equilibrando os tons escuros com pontos de destaque. Alinhando o Caminho: Aprendi a utilizar o Flexbox e Grid para organizar os elementos, garantindo que o conteúdo flua de forma organizada para quem lê, assim como o Wander percorre o terreno em busca dos gigantes. Refinamento Visual: Ajustei margens e espaçamentos, tratando cada parte do site como se eu estivesse desvendando um detalhe escondido nas ruínas de um antigo templo.
